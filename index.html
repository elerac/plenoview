OpenEXR Viewer

A browser-based viewer for OpenEXR files with tev-like navigation, exact pixel values, channels, metadata, ROI stats, panoramas, and export tools.

Open Web App

Local files are processed in your browser. Gallery samples are fetched from their source URLs.

Built for inspection, analysis, and iteration.

Tev-like Navigation

Pan, zoom, and inspect pixels with precise coordinates and value readout.

Channels & Views

Browse RGBA, auxiliary channels, spectral data, Stokes layers, and grouped views.

ROI Statistics

Define regions and compute min, max, mean, and valid sample counts.

Rich Metadata

View header metadata, channels, data windows, and compatible custom attributes.

Panoramas

Preview equirectangular HDRIs with camera controls and lat-long cursor readout.

Export Tools

Export images, screenshots, batches, colormap LUTs, and reproduction metadata.

Technical Details

Formats

  • OpenEXR 2.x
  • Half, float, uint
  • Scanline images
  • Common ZIP and PIZ samples

Viewer

  • Vite + TypeScript
  • WebGL2 renderer
  • exrs WASM decoder
  • Static-hosting ready

Privacy

  • No uploads
  • No tracking
  • Local files stay on your machine
  • Gallery samples fetch remote files

Data Integrity

Raw probe and ROI values stay separate from exposure, gamma, and colormap display transforms.
