Prismifold

OpenEXR viewer for channel-heavy computational images. Inspect local files with raw probes, channel views, ROI statistics, panoramas, and PNG export.

WebGL2 + exrs WASM. Local files stay in your browser.

Open Web App
  • OpenEXR 2.x scanline
  • half / float / uint
  • WebGL2
  • exrs WASM
  • local files stay local

Features

Tools for inspecting, visualizing, measuring, and exporting channel-heavy EXR data.

Inspect

  • Open EXR files from disk, drag and drop, or sample scenes.
  • Probe raw source pixel values before display transforms.
  • Review EXR metadata, channel lists, and image windows.

Visualize

  • Browse RGB, alpha, AOV, spectral, Stokes, and grouped channels.
  • Apply colormaps, exposure, gamma, range, and zero-center controls.
  • Switch between image, panorama, and depth viewing modes.

Measure

  • Draw ROIs and inspect bounds, size, and valid sample counts.
  • Read min, mean, and max statistics for active channel selections.
  • Use rulers and image stats while zooming into exact pixels.

Export

  • Save PNG images with the current display settings applied.
  • Export screenshots, batches, and reproduction metadata.
  • Generate standalone PNG gradients for registered colormaps.

Gallery

Embedded examples stay inspectable on the page and can open into the full viewer.
